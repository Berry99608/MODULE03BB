Ce projet est un jeu du pendu développé en Python.  
Le joueur doit deviner un mot choisi aléatoirement par la machine.
Fonctionnalités

- Sélection aléatoire d’un mot depuis un fichier `mots_pendu.txt`
- Suppression automatique des accents (é → e, à → a, etc.)
- 6 chances par partie
- Affichage du mot avec `_` pour les lettres non trouvées
- Indice donné à la dernière chance
- Système de victoire / défaite
- Possibilité de rejouer
